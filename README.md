# classic-snake-turtle
A classic Snake game built using Python Turtle with collision detection, scoring system, and dynamic food mechanics

---
## Features

- Snake movement in 4 directions
- Food generation with random positions
- Snake growth system
- Score tracking and high score system
- Collision detection (walls & self)
- Game reset mechanics

---

## Controls

- `W` → Move Up  
- `S` → Move Down  
- `A` → Move Left  
- `D` → Move Right  

---

## Technologies Used

- Python 
- Turtle Graphics (built-in)
- Random module
- Time module
